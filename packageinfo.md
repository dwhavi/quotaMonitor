# 기능

- GLM API 잔여 사용량을 VSCode 상태 표시줄에 표시
- `glm-quota-monitor.showQuota` 명령으로 확인
- `glm-quota-monitor.setApiKey` 명령으로 API 키 설정
- `glm-quota-monitor.refresh` 명령으로 새로고침
